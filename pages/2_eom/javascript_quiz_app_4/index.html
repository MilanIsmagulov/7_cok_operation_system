Логотип ОС WINDOWS

(2012-2021 гг.)

Windows – это современная операционная система, которая работает на потребительских ПК, ноутбуках, планшетах и телефонах, а также настольных ПК для бизнеса и корпоративных серверах. Windows также является операционной системой, используемой в игровой системе Microsoft Xbox.

Разработку Microsoft операционной системы Windows для компьютеров на базе ПК, а также серверов можно разделить на четыре эпохи: MS-DOS, MS-DOS-based, Windows, Windows на базе NT (New Technology) и современная Windows. Технически каждая из этих систем существенно отличается от других. Каждая из них доминировала во время разных десятилетий в истории персонального компьютера.


Согласно статистике, Windows является самой популярной операционной системой для настольных компьютеров. И это не случайно, компания Microsoft изначально хотела создать универсальный продукт – ОС, которая будет работать как на самых простых и непроизводительных компьютерах, так и на самых дорогих и мощных. Это очень существенное преимущество ОС Windows, поскольку разработчиков, которые пишут программы для этой операционной системы, намного больше, чем для других платформ.
