Основы теории операционных систем

Ключевые слова

Виртуальная память
процесс, посредством которого данные (например, программный код) могут быстро обмениваться между местами хранения физической памяти и оперативной памятью

Операционная система реального времени
специализированная операционная система, назначение которой – предоставление необходимого и достаточного набора функций для проектирования, разработки и функционирования систем реального времени на конкретном аппаратном оборудовании

Диспетчер памяти
часть операционной системы, обрабатывающая запросы на выделение и освобождение оперативной памяти

Диспетчер устройств
универсальная утилита, с помощью которой можно проверить количество подключенных устройств, их основные типы и правильность подключения

Диспетчер задач
компьютерная программа, отображающая список запущенных процессов и ресурсов, которые они потребляют

Диспетчер файлов или файловый менеджер
предоставляют интерфейс пользователя для работы с файловой системой и файлами

Аппаратное обеспечение
центральный процессор, память, система шин, устройство ввода, например, клавиатура, устройство вывода, например, монитор

Рекомендации для обучающегося (самостоятельная работа)

  • Просмотрите видеолекцию по теме «Основы теории операционных систем».
  • Если в процессе изучения материала возникнут вопросы, запишите их для дальнейшего обсуждения с преподавателем и/или с группой обучающихся.
  • При необходимости прослушайте/просмотрите материал еще раз.
    Возвращайтесь к наиболее сложным аспектам темы.
  • Соблюдайте здоровьесберегающий режим: чередуйте работу с электронными носителями с отдыхом и гимнастикой для глаз.

Рекомендации для обучающегося (самостоятельная работа)

  • Запустите интерактивный тренажер по теме «Основы теории операционных систем».
  • Последовательно ознакомьтесь с условиями учебной задачи.
    Выполните задания интерактивного тренажера. Обратите внимание на отсутствие подсказок и ограниченное число попыток.
  • Выполните тренировочные задания на проверку приобретенных знаний, умений и навыков:
    - интерактивное задание с выбором одного правильного ответа;
    - интерактивное задание с выбором нескольких правильных ответов.
  • Сформулируйте конечный результат своей работы.
    При возникновении ошибок возвращайтесь к повторному изучению теоретического материала.
  • Осуществите самопроверку и предварительную оценку.
    Используйте технологию пошагового контроля по результату за каждое задание.

Рекомендации для обучающегося (самостоятельная работа)

  • Запустите интерактивный тренажер
    Последовательно ознакомьтесь с условиями учебной задачи.
  • Выполните задания.
    Обратите внимание на отсутствие подсказок и ограниченное число попыток.
  • Осуществите самопроверку и предварительную оценку.

    Используйте технологию пошагового контроля по результату за каждое задание.

    При возникновении ошибок возвращайтесь к повторному изучению теоретического материала.
  • Выполните тренировочные задания на проверку приобретенных знаний, умений и навыков (выбрать из списка, указать количество):
    - интерактивное задание с выбором одного правильного ответа;
    - интерактивное задание с выбором нескольких правильных ответов;
    - интерактивное задание с выбором элементов из списков;
    - интерактивное задание с вводом строки;
    - интерактивное задание на установление соответствия;
    - интерактивное задание на распределение по группам (классификацию);
    - интерактивное задание на упорядочивание (выстраивание по порядку);
    - интерактивное задание на установление связей;
    - интерактивное задание на обозначение объектов на рисунке;
    - интерактивное задание на перемещение объектов по экрану.
  • Сформулируйте конечный результат своей работы.

    При возникновении ошибок возвращайтесь к повторному изучению теоретического материала.
